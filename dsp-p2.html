Filtro de Kalman

Es un algoritmo recursivo desarrollado por Rudolf E. Kalman en 1960 que permite estimar el estado oculto de un sistema dinámico lineal, cuando este está sometido a ruido blanco aditivo, minimazando el error cuadrático medio.

Filtro de Kalman - Aplicaciones

Existen numerosas aplicaciones, pero principalmente se utiliza para:

  • Seguimiento de objetos
  • Economía
  • Navegación
  • Visión artificial
  • Fusión de sensores

Filtro de Kalman - Características

  • Se utiliza en sistemas:
    • Discretos
    • Lineales
    • Con ruido blanco aditivo
  • Permite la fusión de sensores.
  • Solo utiliza:
      • Módelo del sistema
      • Estado previo
      • Incertidumbre del modelo
      • Mediciones actuales
      • Incertidumbre de la medición

Filtro de Kalman - Sistema LTI

Continua Discreta
$$ \left\{\begin{aligned} \dot{\mathbf{x}}_{(t)} &= A \mathbf{x}_{(t)} + B \mathbf{u}_{(t)} \\ \mathbf{y}_{(t)} &= C \mathbf{x}_{(t)} + D \mathbf{u}_{(t)} \end{aligned} \right. $$ $$ \left\{\begin{aligned} \mathbf{x}_{[k+1]} &= A \mathbf{x}_{[k]} + B \mathbf{u}_{[k]} \\ \mathbf{y}_{[k]} &= C \mathbf{x}_{[k]} + D \mathbf{u}_{[k]} \end{aligned} \right. $$

A esto se lo denomina Espacio de estados

Filtro de Kalman - Sistema LTI

$\mathbf{x} \quad$ es el Vector de Estado

$\mathbf{y} \quad$ es el Vector de Salida

$\mathbf{u} \quad$ es el Vector de contro

$A \quad$ es la matriz de estados

$B \quad$ es la matriz de entrada

$C \quad$ es la matriz de salida

$D \quad$ es la matriz de Transmisión Directa
